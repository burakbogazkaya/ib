# Deezer Romania Network Analysis

This project analyzes the Deezer Romania friendship network using Python and NetworkX.

## Contents
- Network construction from edge list
- Degree distribution analysis
- Connected components
- Approximate shortest path length
- Clustering coefficient and density
- Centrality analysis
- Comparison with ER, BA, and WS synthetic graphs
- Linear Threshold diffusion experiments
- Greedy seed selection

## Main files
- `data_driven_strategic_decision_making_project.py`
- `degree_distribution_romania.png`

## Dataset
The project uses the Deezer European friendship network dataset, focusing on the Romania subnetwork.
